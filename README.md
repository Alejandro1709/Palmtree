## Run Locally

Clone the project

```bash
  git clone https://github.com/Alejandro1709/Palmtree.git
```

Go to the project directory

```bash
  cd Palmtree
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
